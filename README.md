## 📌 About the Project  

**Tic Tac Toe Game** is a classic two-player strategy game built using **HTML, CSS, and JavaScript**. Players take turns marking X and O on a 3×3 grid. The first player to align three symbols horizontally, vertically, or diagonally wins the game.

This project demonstrates fundamental front-end development concepts including DOM manipulation, event handling, conditional logic, and game state management.

---

## 🚀 Features  

- ❌⭕ Two-player turn-based gameplay  
- 🏆 Win detection logic  
- 🤝 Draw detection  
- 🔄 Reset and New Game functionality  
- 🔊 Sound effects for moves and results  
- 📱 Responsive design
